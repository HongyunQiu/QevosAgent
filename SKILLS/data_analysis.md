# SKILL: 数据分析

适用领域：数据处理、统计分析、可视化、机器学习、ETL。

## 分析流程

1. 数据探查：先了解数据量级、字段类型、缺失值分布
2. 清洗优先：处理空值、异常值、重复行后再做分析
3. 逐步验证：每一步用 `print(df.head())` / `df.describe()` 确认结果
4. 结论先行：分析完先写结论，再附图表支撑

## 代码规范

- 使用 pandas/numpy 进行数据处理，避免手写循环
- 大文件（>50MB）用 `chunksize` 分块读取
- 图表用 matplotlib/seaborn，保存为 `$RUN_DIR/artifacts/图表名.png`
- 模型训练结果写入 `$RUN_DIR/artifacts/model_report.json`

## 数据安全

- 不在日志或输出中打印完整原始数据行（超过 10 行要截断）
- 含 PII 字段（姓名/手机/身份证）需脱敏后展示

## 输出标准

- 每次分析附上：数据规模、关键指标、发现摘要
- 图表必须有标题、坐标轴标签、图例
