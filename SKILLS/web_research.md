# SKILL: 网络调研

适用领域：技术调研、竞品分析、文档检索、信息汇总。

## 调研策略

1. 明确目标：先在草稿本写下"需要回答的核心问题"，再开始搜索
2. 多关键词：同一问题用 3 种以上关键词组合搜索，避免偏差
3. 来源权重：官方文档 > 技术博客 > 问答社区 > 第三方文章
4. 时效优先：优先查看近 2 年内容；旧内容需注明版本/年份

## 信息整理

- 每条关键发现写入草稿本，格式：`- [来源简称] 结论`
- 互相矛盾的信息要标注，不要随意取舍
- 最终汇总时注明信息来源和获取时间

## 工具使用

- `web_search`：关键词检索，每次搜索不超过 5 个结果，但可多次搜索
- `shell` + curl：直接获取技术文档原文
- `analyze_content`：对多篇长文档联合分析

## 输出格式

结论报告结构：
1. 核心问题回答（1-3 句）
2. 关键发现列表（带来源）
3. 不确定/存疑点
4. 建议后续行动
